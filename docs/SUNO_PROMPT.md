## STYLE DESCRIPTION (Custom Mode — paste into Style field)

```
Intimate solo piano, ambient electronic textures, emotional cinematic build, 90BPM, key of C major, warm analog feel, progressive dynamics, deeply personal not epic, not corporate. Ólafur Arnalds meets Nils Frahm. Felt piano with subtle room reverb, sparse string ensemble entering mid-piece, warm analog synth pad, no percussion until final act. Earthy, organic, human. - **Full:** 1:15-2:00 (strings, community, warmth)
```

## REFERENCE TRACKS (listen before generating)

1. **Ólafur Arnalds — "Near Light"** — The gold standard for intimate piano + ambient
2. **Nils Frahm — "Says"** — Build structure, organic warmth
3. **Hania Rani — "Glass"** — Sparse, emotional, room sound

---

## GENERATION STRATEGY

1. **Generate 3-4 variants** with the full prompt above
2. **Pick the best "Intro→Chorus" section** — this is 80% of the emotional impact
3. **Use Extend** to build out the Bridge and Outro if the first generation is promising
4. **If piano feels too "clean":** Add "felt piano, slightly out of tune, room ambience, close mic" to style
5. **If build feels too corporate:** Add "not epic, not triumphant, intimate, small room" to style and exclude "orchestral, triumphant, heroic"
